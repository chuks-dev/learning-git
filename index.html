Learning git

  1. To initialize git run:

    git init

  2. To see git status run:

    git status

  3. To track files run :

    git add --filename-- 
    note: you can add all the files in a directory by using a dot(.) instead of the file names

  4. To restore modified files to previous add, run:

    git restore --filename--

  5. To commit, make sure all your files are being tracked by adding them(see NO. 3), then run:

    git commit -m "--your message--" 
    the -m flag stands for message

  6. To ignore a file or repository, create a .gitignore file, then add the path to the files you wish to ignore

Pushing files to github

  1. Create a github account

  2. Create a repository in you github account

  3. Add you repository to your local github repo by running

    git add origin master --repositoryURL--

  4. Merge your current directory to your master

    git merge --branch--

  5. Commit your changes and push them to github by running

    git push origin --master--
